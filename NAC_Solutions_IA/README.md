# 📊 NAC Solutions — Análise de Disparidade de Preços da Cesta Básica do Uíge

Sistema inteligente de análise e previsão da disparidade de preços da cesta básica na Província do Uíge, Angola, utilizando Machine Learning.

## 🚀 Aceder à Aplicação

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://nac-solutions-cesta-basica.streamlit.app)

## 📌 Sobre o Projecto

- **997 registos** recolhidos em campo em Abril de 2026
- **8 mercados** analisados na Província do Uíge
- **3 algoritmos** de Machine Learning implementados

## 🤖 Algoritmos de IA

| Algoritmo | Função |
|-----------|--------|
| Random Forest | Previsão de preços por mercado |
| K-Means | Agrupamento de mercados (Barato / Médio / Caro) |
| Isolation Forest | Detecção automática de preços anómalos |

## 🛠️ Stack Tecnológico

Python · Streamlit · Pandas · Scikit-learn · Plotly · NumPy · OpenPyXL
